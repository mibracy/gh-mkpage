# MkDocs-Material Demo

Expert guidance for building documentation sites with Material for MkDocs. Covers configuration, content patterns, theme customization, and deployment.

**Capabilities:**
1. Full `mkdocs.yml` scaffolding with theme, nav, palette, extensions
2. Navigation pattern selection (tabs, sections, indexes, instant)
3. Admonition, tabs, annotation, Mermaid diagram content patterns
4. Plugin setup (search, social, blog, offline, privacy)
5. Troubleshooting common issues (Mermaid rendering, palette toggles, mobile nav)

## ROI to human

Accelerate docs iteration and eliminate manual config lookup. Treats mkdocs-material as a first-class platform with turnkey snippets.
