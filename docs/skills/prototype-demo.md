# Prototype Demo

Build throwaway code to answer a question. Two branches:

- **Logic** — interactive terminal app to push a state machine through edge cases
- **UI** — multiple radically different UI variations switchable via URL param + bottom bar

**Usage:** "prototype this data model", "try a few designs for this page"

Tip: pick the branch based on the question being answered, not the format preference.

## ROI to human

Answers design questions before committing to code. Fail fast on paper, not in prod.
