# Zoom-Out Demo

Asks the agent to go up a layer of abstraction and provide broader context.

**Usage:** "zoom out on this code", "I don't understand this area", "how does this fit in the bigger picture"

Returns a map of relevant modules and callers using the project's domain glossary.

## ROI to human

Get your bearings fast. No spelunking through 20 files to understand one module.
